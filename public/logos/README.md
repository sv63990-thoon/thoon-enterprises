# Thoon Logo Files

This directory contains the optimized Thoon Enterprises logo files for web use.

## Files
- `thoon_logo_web.png` - Standard web logo (PNG)
- `thoon_logo_web_2x.png` - High-resolution logo (Retina/2x screens)
- `thoon_logo_web.svg` - Scalable SVG version (best for modern websites)

## Usage
These logos are optimized for glass UI backgrounds and have transparent backgrounds with balanced padding for navbar alignment.
